CSE 274 Midterm - AlternatingSignRearranger
View Project Code on GitHub

This was my timed Midterm assessment (80 minutes). AlternatingSignRearranger takes an array with both positive and negative integers and adds them to separate LinkedLists (positive/negative). While maintaining O(1) time complexity, the program rearranges the integer values back into a single LinkedList so that the values are alternating between positive/negative, and the negative values are now listed in the opposite order as how they were first entered.
